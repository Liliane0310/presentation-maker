# presentation-maker

一个专为「讲给人听」设计的 Claude 技能：一键生成**教学科普型、自包含 HTML 演示稿**，无需联网、不依赖外部资源，打开即用。

✨ 特性亮点：
- 📄 单个 `.html` 文件（双击就能看，支持手机/电脑）
- 🎤 内置演讲者模式（按 `S` 键开启，含计时器+逐字稿+下一页预览）
- ⌨️ 全键盘操作：`←→` 翻页、`Home/End` 跳首尾、`F` 全屏
- 🌟 温暖纸感设计：小赖字体 + 暖色系 + 微纹理背景，专注内容不刺眼
- 🗣️ Notes 专为「讲出来」优化（不是写论文，是写说话节奏）

📁 本包已包含：
- `SKILL.md`：完整行为规范与工作流  
- `references/`：视觉系统、页面结构规则、讲稿写法指南  
- `examples/`：可直接打开参考的成品样例（如 `什么是 Skill.html`）

🎯 适合场景：新人培训、技术分享、产品讲解、课堂教案、内部汇报。
